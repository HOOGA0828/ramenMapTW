# 台灣拉麵地圖

一個以台灣拉麵店為主題的互動式地圖網站，讓使用者可以快速探索附近或特定區域的拉麵店，並透過投稿機制補充更多在地店家資訊。

## 專案特色

- 互動式地圖瀏覽：使用地圖介面呈現全台拉麵店分布，支援縮放、定位與店家資訊查看。
- 店家搜尋與篩選：可依店名、地區與拉麵風格快速找到想看的店家。
- 店家投稿入口：使用者可以提交新店家資料，協助補完整體地圖資料。
- 後台審核流程：投稿與候選店家可經由管理介面審核，降低重複與錯誤資料。
- 地圖資料整合：專案可整合開放地圖資料與人工補充資料，逐步建立更完整的台灣拉麵資料集。
- 行動裝置友善：介面以手機與桌面瀏覽都能使用為目標設計。

## 技術棧

- Next.js App Router
- React
- TypeScript
- MapLibre GL JS
- Supabase
- Vercel

## 主要頁面

- `/`：拉麵地圖與店家列表。
- `/submit`：店家投稿表單。
- `/admin`：店家與投稿審核後台。

## 專案目標

台灣有許多散落在巷弄、社群與地圖評論中的拉麵店資訊。這個專案希望把它們整理成更容易探索的地圖體驗，讓喜歡拉麵的人可以更直覺地發現新店，也讓資料能透過投稿與審核持續改善。

## 狀態

目前專案仍在持續整理資料與改善體驗中。
